# Ejercicios de Lógica en JavaScript para Entrevistas de Trabajo

¡Bienvenido! Este repositorio contiene una serie de ejercicios prácticos diseñados para ayudarte a practicar y poner a prueba tus habilidades lógicas en JavaScript. Estos son problemas clásicos y recurrentes en entrevistas técnicas de desarrollo de software.

Cada archivo contiene el mandato detallado en la parte superior, una estructura inicial de la función que debes implementar y un conjunto de pruebas/casos de prueba al final para verificar tu solución de manera automática.

## Cómo Ejecutar y Probar tus Soluciones

Para probar cada ejercicio, asegúrate de tener instalado **Node.js** en tu computadora.

1. Abre tu terminal en esta carpeta (`c:\Users\luisl\Desktop\Ejercicios de prueba logicas\Javascript`).
2. Ejecuta el archivo que deseas probar con el comando `node`:
   ```bash
   node 01_fizzbuzz.js
   ```
3. Si la consola muestra mensajes de éxito o no lanza errores en los `console.assert`, significa que tu lógica es correcta. Si falla un caso, verás un error indicando qué prueba falló.

¡Mucho éxito con tu preparación!
